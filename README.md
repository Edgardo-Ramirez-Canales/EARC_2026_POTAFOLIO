# Portafolio Personal — Edgardo Ramírez Canales

Sitio web de portafolio personal desarrollado con HTML, CSS y JavaScript vanilla en un único archivo autocontenido.

## Descripción

Portafolio profesional que presenta habilidades, proyectos y datos de contacto de Edgardo Ramírez, Ingeniero en Sistemas. Diseño dark mode con tipografías Inter y Fira Code.

## Tecnologías

- HTML5
- CSS3 (variables, grid, flexbox, animaciones)
- JavaScript vanilla
- Google Fonts (Inter, Fira Code)

## Estructura del proyecto

```
Proyecto_1_2025_EARC/
└── portfolio_edgardo_2.0.html   # Archivo principal del sitio
```

## Uso

Abrir el archivo `portfolio_edgardo_2.0.html` directamente en cualquier navegador moderno. No requiere servidor ni dependencias.
